large scale .txt file management



These are both C++ programs that showcase Data Structures that are commonly used in software engineering.

The card game uses a stack just as if you were at a casino. The dealer can only take from the top of the deck.

The notepad program can read several .txt files of data and manipulate the data without losing values. In the example I used 3 football seasons and merged/sorted them while maintaining the associated scores. Both programs heavily rely on the use of pointers and structs.
